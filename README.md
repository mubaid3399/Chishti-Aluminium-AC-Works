# Steelcrest

Premium Aluminium & Steel Solutions website built with React, Vite, TailwindCSS, and shadcn/ui.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm 9+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens at [http://localhost:5173](http://localhost:5173)

### Production Build

```bash
npm run build
npm run preview
```

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite 6
- **Styling:** Tailwind CSS 4
- **UI Components:** shadcn/ui (Radix UI)
- **Routing:** Wouter
- **State Management:** TanStack React Query
- **Icons:** Lucide React, React Icons
- **Animations:** Framer Motion
